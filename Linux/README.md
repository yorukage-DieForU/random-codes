# Linux

There was a time when I only wanted things to work.  
Now I want to understand what is behind them.

But maybe this journey was never only about Linux.

Every command I learn, every problem I solve, and every night spent trying to understand something reminds me that life works the same way. We break, we fail, we search for answers, and sometimes we have to rebuild ourselves from the pieces that remain.

Some people leave. Some dreams disappear. Some versions of ourselves never come back.

But Christ remains.

This folder is a record of my journey: the things I learn, the mistakes I make, and the small steps I take while trying to become someone better than I was yesterday.

> "The person I was had to disappear so the person God is creating could be born."

God bless us.
